## 一、🎨 素材库：一站式资源管理中心

### 1.1 功能亮点

全新的素材库模块，让大屏制作不再为寻找背景、图标、插画而烦恼：

| 分类 | 用途 |
|------|------|
| **背景** | 大屏底图、渐变背景 |
| **面板** | 数据展示容器、装饰框 |
| **图标** | 功能图标、状态指示 |
| **插画** | 装饰元素、场景图 |

支持 **公开素材** 与 **私有素材** 两种可见性，团队协作更灵活！

## 二、📄 多画布：一个项目，多页呈现

### 2.1 告别单页限制

以前一个大屏项目只能做一页？现在 **Forge Report Studio 支持多画布编排**！

通过左侧「页面管理」面板，你可以：

| 操作 | 说明 |
|------|------|
| **新建页面** | 创建普通页面或弹窗页面 |
| **设置首页** | 指定默认展示页 |
| **页面排序** | 拖拽调整显示顺序 |
| **复制页面** | 快速复用已有设计 |
| **重命名/删除** | 灵活管理页面结构 |

### 2.2 页面类型：普通页 vs 弹窗页

每个画布可以选择 **页面类型**：

| 类型 | 用途 | 特点 |
|------|------|------|
| **普通页** | 大屏主页面 | 可设为首页，支持页面跳转 |
| **弹窗页** | 详情弹窗、数据钻取 | 支持动画、遮罩、按钮配置 |

弹窗页的设计让 **数据钻取** 变得前所未有的简单！

## 三、🪟 弹窗系统：交互体验再升级

### 3.1 弹窗配置一览

弹窗页支持丰富的配置项，满足各种业务场景：

**基础配置：**
- 📐 尺寸：宽度、高度、自适应高度
- 📍 位置：居中 / 右侧抽屉 / 底部抽屉
- 🎨 主题：`screen` / `glass` / `flat` / `neon` 等 **9 种主题风格**
- ✨ 动画：缩放 / 右滑入 / 上滑入 / 淡入

**遮罩与关闭：**
- 遮罩透明度可调
- 点击遮罩是否关闭
- 关闭按钮显隐

**底部按钮：**
- 取消/确认按钮自定义文本
- 按钮动作：关闭弹窗 / 跳转页面 / 打开另一个弹窗 / 提交请求

### 3.2 组件触发弹窗

更强大的是，**任何图表组件都可以配置事件来触发弹窗**！

例如：
- 点击柱状图某根柱子 → 打开详情弹窗
- 点击表格某行 → 弹窗展示完整数据
- 弹窗内点击按钮 → 打开另一个弹窗（弹窗套弹窗）

**上下文自动传递**：钻取参数（如点击的数据项 ID）会自动传递给弹窗页，实现精准数据钻取！

## 四、🛠️ 技术架构一览

对于开发者，以下是三大功能的核心技术架构：

```
素材库模块
├── 素材管理页（views/project/materials）
├── 素材选择器组件（MaterialAssetSelector）
├── 图片组件适配器（materialAssetAdapter）
└── API 层：分页查询、上传、删除、重命名

多画布模块
├── 页面管理面板（ContentPages）
├── 中央状态管理（chartEditStore）
├── 页面工具函数（reportPages.ts）
└── 数据结构：ReportMultiPageStorage

弹窗系统
├── 弹窗配置面板（CanvasPage/index.vue）
├── 弹窗运行时管理（modalStack）
├── 组件事件钩子（useLifeHandler）
└── 支持：打开、关闭、上下文传递
```
