- [x] 外层 `designer-nav` 默认收起，并保留图标导航和展开按钮。
- [x] 左右面板收起控制移入各自模块，顶部移除左右箭头。
- [x] “按字段生成 / 清理失效字段”移入更多菜单，更多菜单放在工具栏最右侧。
- [x] “清空画布”按钮文案改为“清空”。
- [x] 左右折叠箭头按钮尺寸、位置和 padding 统一。
- [x] 更多按钮改为高对比实心图标按钮。
- [x] 左侧展开态收起按钮移入组件库标题栏 action 区，不再覆盖标题。
- [x] 折叠态展开按钮改为水平居中，避免左右 padding 视觉不一致。
- [x] 工具栏更多菜单右侧按钮改为控制右侧属性栏，右侧收起时不再保留模块内展开按钮和空白窄栏。
- [x] 主表单画布标题区压缩为单行展示，并将“补齐未使用字段 / 旧版画布”透传到新版画布“按字段生成”同一个更多操作里。
- [x] 选中新版画布中的表单组件时自动展开右侧配置面板。
- [x] 顶部保存按钮 loading 前置，避免保存前同步转换导致用户误以为卡顿。
- [x] 画布新增浮动缩放控制，支持按钮、下拉和 `Ctrl/⌘ + 滚轮` 缩放。
- [x] 将画布缩放控制收纳为底部图标按钮，点击后弹出缩放输入，减少常驻面板占位。
- [x] 表单画布拆分外层固定视口和内部滚动层，缩放控件固定在画布右下角，不再随内容滚动丢失。
- [x] 布局组件结构槽位不再显示复制/删除浮层，避免“拖入字段或布局”空槽被误删。
- [x] 栅格、表格、卡片、标签页、折叠面板的内部投放区统一接管拖拽事件，减少拖入组件时偶发不生效。
- [x] 降低布局组件多层边框和空占位边框，保留选中态与投放态反馈。
- [x] 修复拖入布局空槽时外层容器误报“不支持”的问题，栅格/表格会按鼠标位置选择最近内部槽位。
- [x] 非法拖拽提示改为悬浮提示，不再挤压画布导致拖拽时屏幕抖动。
- [x] 布局容器默认隐藏大部分设计边框，仅在 hover、选中和投放状态显示边界。
- [x] 布局组件选中态补强，栅格、表格、卡片、标签页、折叠面板选中后使用清晰蓝色边框和阴影。
- [x] 表单画布节点 resize 锚点扩展为上、右、下、左和四个角八向锚点，支持在画布上直接调整宽高。
- [x] 栅格/表格内部已有组件时，移动已有组件也按鼠标位置命中内部槽位，避免只能放一个元素的错觉。
- [x] 组件更多菜单增加“隐藏边框”，可关闭默认边框展示，同时不影响选中态边框。
- [x] 结构槽位选中态只保留内部“拖入字段或布局”投放区蓝框，移除外层节点重复选中边框。
