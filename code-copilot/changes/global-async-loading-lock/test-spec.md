# 全局异步 Loading 锁交互测试计划

## P0 验证

- 静态检查：`git diff --check`。
- 前端构建：`source ~/.nvm/nvm.sh && nvm use v20.19.0 && NODE_OPTIONS=--max-old-space-size=8192 pnpm --dir forge-admin-ui build`。

## P1 手工/浏览器验证建议

- 登录后切换菜单，确认全局遮罩出现并在页面加载完成后消失。
- 在列表页点击查询/分页，确认遮罩出现，重复点击被拦截。
- 提交新增/修改/删除，确认成功和失败分支都能关闭遮罩。
- 文件上传成功/失败后遮罩关闭。
- 导出/下载操作过程中出现导出/下载文案。
- 打开含缩略图回显的详情页，确认后台图片渲染请求不会长时间锁定页面。

## 跳过项

- 本批不新增后端逻辑，不执行 Maven 编译。
- 不默认执行全量 E2E，除非本地服务已经可用并需要进一步截图验证。
