# Test Spec

## 本轮增量验证

### P0

- `pnpm exec eslint` 检查对象设计器和动作配置组件。
- `git diff --check` 检查空白和补丁格式。

### P1

- 前端构建：`NODE_OPTIONS=--max-old-space-size=8192 pnpm build`。
- 若本地环境耗时过高或依赖状态阻塞，记录跳过原因。

## 验证重点

- 普通 UI 不展示内部 DSL 类型值。
- 字段映射下拉能使用主表字段、子表字段、当前行字段。
- JSON 高级配置仍可折叠查看和兜底编辑。
- 搜索条件输入框不展示字数统计。
- 列表设计器顶部能直接进入自定义操作配置。
- 关系类型只展示“包含多个明细”。
- 删除最后一个自定义操作后配置弹窗仍保留空状态和添加入口。
- 业务流程配置不再出现编号生成配置步骤。
- 单据基础配置发起人固定为创建人，不出现负责人选择。
- 业务记录绑定不出现负责人字段。
