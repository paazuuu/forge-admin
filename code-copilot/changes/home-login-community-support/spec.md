# 首页社区支持与登录页性能优化

## 背景

用户反馈登录页动效在部分设备上会卡顿，需要去掉无实际业务价值的持续动画。同时首页需要在明显位置展示 ForgeAdmin 交流群二维码，并以更委婉的方式展示维护支持收款码，吸引用户加入社区和支持项目维护。

## 范围

- 登录页去除高成本持续动画、扫光、流光边框、装饰圆点和入场动画，保留静态背景与必要交互反馈。
- 登录页采用左侧较大、右侧较小的静态产品视觉布局；右侧登录区只保留必要文案，尽量减少英文和无业务价值说明。
- 登录页采用全屏左右分栏，`login-brand` 和 `login-form-wrapper` 直接铺满视口，避免居中卡片压缩导致表单或轮播看不全。
- 登录页需要保持紧凑并兼容小屏：小屏下隐藏左侧视觉，只保留登录表单，页面允许竖向滚动。
- 首页统计卡片下方新增“ForgeAdmin 社区”区域。
- 展示交流群二维码和维护支持二维码，二维码区域需要足够明显。
- 二维码支持点击放大预览。
- 收款码文案采用“支持维护 / 请维护者喝杯咖啡 / 随心支持”等委婉表达。

## 非目标

- 不调整登录认证逻辑、验证码逻辑、租户逻辑。
- 不改首页待办、公告、快捷入口、图表的数据逻辑。
