## 验收点

- 新建模型时默认能看到 `id` 和审计字段，且不能编辑、删除。
- 点击“保存配置”只保存 `ai_lowcode_model.model_schema`，不执行 DDL。
- 点击“保存并同步表结构”保存配置后执行受控 DDL，新表含 `id` 自增主键，已有表只追加缺失业务字段。
- 应用设计页不再承担字段定义或 DDL 同步。
- 自定义按钮能配置到工具栏或行操作列，运行预览中可见。
- 发布页不再要求用户选择建表方式。
